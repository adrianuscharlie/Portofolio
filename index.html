Minum Obat: Daily Reminder Medicine Treatment

Minum Obat is an App that design both for pharmacist and patient. This app can input some medical prescription and set daily reminder to consume the medicine for patient that have been inputed by the pharmacist. For the patient, they can get the daily treatment notification from the app and can look for their schedule and the detail of the medicine.This application is made using the Flutter programming language and uses Firebase as a backend
